# Documentation Index

Each file answers one part of the project instead of mixing theory, code, and
results in a single long README.

- [02_research_background.md](02_research_background.md): research motivation, task definition, labels, and ethical framing.

- [04_models.md](04_models.md): MentalRoBERTa, traditional baselines, and fusion architectures.
- [05_training_evaluation.md](05_training_evaluation.md): commands, metrics, validation/test protocol, and output paths.
- [06_results_summary.md](06_results_summary.md): current result snapshot and how to read model outputs.
